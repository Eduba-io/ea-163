# Hedgehog x Eduba landing page

A bespoke, prospect-facing landing page prepared by Eduba for Hedgehog (hedgehog.cloud). Built for booth handoff at the conference: Matt opens the page on his phone at booth 222 and walks Marc Austin or Mike Dvorkin through the 60 / 30 / 10 read and the ICM paper.

Intended URL path: /for/hedgehog

## Files

- `index.html` — single-page landing, inline SVG diagram, no JS
- `styles.css` — Hedgehog brand tokens (gold on charcoal, Inter, warm cream surface)
- `logo.svg` — Hedgehog wordmark, copied from Brand Pack assets

## Notes

- Only brand-pack.json tokens used for color, type, radius, shadow.
- Inter self-hosted via rsms.me (the de facto engineering-brand font that matches Hedgehog's rendered HubSpot stack).
- No Eduba branding above the footer. Eduba appears once, in the footer, as "Prepared by Eduba / eduba.io."
- CTA points at Matt Creamer: https://calendly.com/thecro-eduba/30min.
- No pricing surfaced. Scope language only ("Scoped sprint," "Workshop + advisory + OCA").
- Every URL is a real `<a href target="_blank" rel="noopener">` anchor.
- Mobile verified at 375px breakpoint.
